# react-native-transact-sdk

React Native bridge for the Transact SDK

## Installation

```sh
npm install react-native-transact-sdk
```

## Usage

```js
import { multiply } from "react-native-transact-sdk";

// ...

const result = await multiply(3, 7);
```
